FAQ

What is WinTils?

WinTils is a Windows 11 quality-of-life toolkit that automates safe debloating, performance tuning, and personalization.

How do I run it?

Open PowerShell as Administrator and run the one‑liner above to fetch and launch the latest script.

Troubleshooting: PowerShell blocked this script

Allow scripts for this session only and retry:

Set-ExecutionPolicy -Scope Process -ExecutionPolicy Bypass
Can I review the code first?

Yes—open the raw script in your browser:

https://raw.githubusercontent.com/KrexonTX/WinTils/refs/heads/main/main.ps1
How do I revert changes?

Create a restore point before major changes. Many toggles are reversible; consult logs or re-run with restore options when provided.
